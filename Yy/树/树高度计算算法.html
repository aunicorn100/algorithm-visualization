树的高度计算

以孩子链存储结构实现树高度计算的递归算法，支持单步/自动演示

算法思路

树的高度计算递归思想：

  • 空树高度为 0
  • 递归计算每个子树高度
  • 树高度 = 最大子树高度 + 1

f(t) = 0 (t=NULL)

f(t) = MAX{f(p)} + 1 (p是子节点)

核心算法代码 


// 求树t高度的递归算法
int TreeHeight1(TSonNode *t)
{
    TSonNode *p;
    int i, h, maxh = 0;
    if (t == NULL) return 0;     // 空树返回高度0
    else                         // 处理非空树
    {
        for (i = 0; i < MaxSons; i++)
        {
            p = t->sons[i];       // 指向第i个孩子
            if (p != NULL)        // 孩子存在
            {
                h = TreeHeight1(p); // 递归计算子树高度
                if (maxh < h) maxh = h; // 更新最大高度
            }
        }
        return (maxh + 1);       // 返回当前树高度
    }
}

复杂度分析

时间复杂度: O(n)

访问每个节点一次

空间复杂度: O(h)

递归栈深度=树高h
