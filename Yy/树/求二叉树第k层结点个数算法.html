例题7.14：求二叉树中第k层的结点个数

基于二叉链存储结构的递归算法实现与可视化（先序遍历思想）

问题描述

假设二叉树采用二叉链存储结构，设计一个算法 Lnodenum() 求二叉树b中第k层的结点个数（根节点为第1层）。

设 Lnodenum(b, h, k, n) 统计二叉树b中第k层的结点个数，其中：

  • h表示b所指结点的层数
  • n为引用型参数，用于保存第k层的结点个数
  • 初始调用：n=0; Lnodenum(b, 1, k, n);

递归模型

f(b, h, k, n) = 直接返回 // 当 b = NULL

f(b, h, k, n) = n++ // 当 h = k（当前结点在目标层）

f(b, h, k, n) = f(b->lchild, h+1, k, n) + f(b->rchild, h+1, k, n) // 当 h < k（继续遍历子树）

算法实现 


void Lnodenum(BTNode *b, int h, int k, int &n)
{
    if (b == NULL)
        return; // 空树直接返回
    else
    {
        if (h == k)
            n++; // 当前结点在第k层，n增1
        else if (h < k)
        {
            Lnodenum(b->lchild, h+1, k, n);
            Lnodenum(b->rchild, h+1, k, n);
        }
    }
}

提示：该算法基于先序遍历的思想，先判断当前结点是否在目标层，若未到目标层则递归遍历左右子树。
