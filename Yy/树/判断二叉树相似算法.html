例题7.15：判断两棵二叉树是否相似

基于二叉链存储结构的递归算法实现与可视化

问题描述

假设二叉树采用二叉链存储结构，设计一个算法判断两棵二叉树是否相似。所谓二叉树b1和b2是相似的，指的是：

  • b1和b2都是空的二叉树；或者
  • b1和b2的根结点是相似的，以及b1的左子树和b2的左子树是相似的且b1的右子树与b2的右子树是相似的。

递归模型

f(t1, t2) = true // 若 t1 = t2 = NULL

f(t1, t2) = false // 若 t1、t2 之一为空，另一不为空

f(t1, t2) = f(t1->lchild, t2->lchild) && f(t1->rchild, t2->rchild) // 其他情况

算法实现 


bool Like(BTNode *b1, BTNode *b2)
// b1和b2两棵二叉树相似时返回true，否则返回false
{
    bool like1, like2;
    if (b1 == NULL && b2 == NULL)
        return true; // 两棵树都为空，相似
    else if (b1 == NULL || b2 == NULL)
        return false; // 一棵树为空，另一棵不为空，不相似
    else
    {
        // 递归判断左子树是否相似
        like1 = Like(b1->lchild, b2->lchild);
        // 递归判断右子树是否相似
        like2 = Like(b1->rchild, b2->rchild);
        // 左右子树都相似，则两棵树相似
        return (like1 && like2);
    }
}
