例题9.7：平衡二叉树(AVL)的插入与旋转调整详解

演示AVL树在插入新节点后，如何通过LL、RR、LR、RL四种旋转操作恢复平衡

AVL树的四种不平衡情况与旋转

LL型不平衡 (右旋转)

在最小不平衡子树的根节点的左孩子的左子树上插入新节点，导致不平衡。

调整方法：对最小不平衡子树的根节点进行一次右旋转。

RR型不平衡 (左旋转)

在最小不平衡子树的根节点的右孩子的右子树上插入新节点，导致不平衡。

调整方法：对最小不平衡子树的根节点进行一次左旋转。

LR型不平衡 (先左后右旋转)

在最小不平衡子树的根节点的左孩子的右子树上插入新节点，导致不平衡。

调整方法：先对左孩子进行左旋转，再对根节点进行右旋转。

RL型不平衡 (先右后左旋转)

在最小不平衡子树的根节点的右孩子的左子树上插入新节点，导致不平衡。

调整方法：先对右孩子进行右旋转，再对根节点进行左旋转。

核心旋转算法

LL型 (右旋转)

// R_Rotate: 对结点p进行右旋转
void R_Rotate(AVLNode *&p) {
    AVLNode *lchild = p->lchild;
    p->lchild = lchild->rchild;
    lchild->rchild = p;
    // 更新平衡因子
    p->bf = 0;
    lchild->bf = 0;
    p = lchild; // p指向新的根
}

RR型 (左旋转)

// L_Rotate: 对结点p进行左旋转
void L_Rotate(AVLNode *&p) {
    AVLNode *rchild = p->rchild;
    p->rchild = rchild->lchild;
    rchild->lchild = p;
    // 更新平衡因子
    p->bf = 0;
    rchild->bf = 0;
    p = rchild; // p指向新的根
}

LR型 (先左后右)

// LeftBalance: 左子树高，需LL或LR旋转
void LeftBalance(AVLNode *&p) {
    AVLNode *lchild = p->lchild;
    switch(lchild->bf) {
        case LH: // LL型
            p->bf = lchild->bf = EH;
            R_Rotate(p); break;
        case RH: { // LR型
            AVLNode *rchild = lchild->rchild;
            switch(rchild->bf) {
                case LH: p->bf = RH; lchild->bf = EH; break;
                case RH: p->bf = EH; lchild->bf = LH; break;
                case EH: p->bf = lchild->bf = EH; break;
            }
            rchild->bf = EH;
            L_Rotate(p->lchild); // 先左
            R_Rotate(p); // 后右
            break;
        }
    }
}

RL型 (先右后左)

// RightBalance: 右子树高，需RR或RL旋转
void RightBalance(AVLNode *&p) {
    AVLNode *rchild = p->rchild;
    switch(rchild->bf) {
        case RH: // RR型
            p->bf = rchild->bf = EH;
            L_Rotate(p); break;
        case LH: { // RL型
            AVLNode *lchild = rchild->lchild;
            switch(lchild->bf) {
                case RH: p->bf = LH; rchild->bf = EH; break;
                case LH: p->bf = EH; rchild->bf = RH; break;
                case EH: p->bf = rchild->bf = EH; break;
            }
            lchild->bf = EH;
            R_Rotate(p->rchild); // 先右
            L_Rotate(p); // 后左
            break;
        }
    }
}
