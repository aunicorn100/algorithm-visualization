例题9.5：二叉排序树(BST)的结点删除操作详解

演示BST中删除叶子结点、单孩子结点及双孩子结点的完整过程

删除操作的四种情况

情况一：删除叶子结点

被删除的结点是叶子结点（无左、右孩子），直接将其从树中移除，并修改其双亲结点的指针为NULL。

情况二：删除仅有左子树的结点

用被删除结点的左孩子结点替换它，即修改其双亲结点的指针，使其指向被删除结点的左孩子。

情况三：删除仅有右子树的结点

用被删除结点的右孩子结点替换它，即修改其双亲结点的指针，使其指向被删除结点的右孩子。

情况四：删除有左右子树的结点

用其中序前驱（左子树中值最大的结点）或中序后继（右子树中值最小的结点）的值替换被删除结点的值，然后删除该前驱或后继结点（该结点必然是叶子结点或仅有单孩子）。

核心删除算法 (DeleteBST)

BSTNode *DeleteBST(BSTNode *bt, KeyType k) {
    if (bt == NULL) return bt; // 树空，直接返回
    BSTNode *p = bt, *f = NULL; // p指向待删结点，f指向其双亲

    // 1. 查找待删除结点 p
    while (p != NULL) {
        if (p->key == k) break; // 找到待删结点
        f = p;
        p = (k < p->key) ? p->lchild : p->rchild;
    }
    if (p == NULL) return bt; // 未找到，树不变

    // 2. 分情况删除结点 p
    if (p->lchild == NULL && p->rchild == NULL) { // 情况1: 叶子结点
        if (p == bt) bt = NULL; // 待删结点是根
        else if (f->lchild == p) f->lchild = NULL;
        else f->rchild = NULL;
        free(p);
    } else if (p->rchild == NULL) { // 情况2: 只有左子树
        if (f == NULL) bt = p->lchild; // 待删结点是根
        else if (f->lchild == p) f->lchild = p->lchild;
        else f->rchild = p->lchild;
        free(p);
    } else if (p->lchild == NULL) { // 情况3: 只有右子树
        if (f == NULL) bt = p->rchild; // 待删结点是根
        else if (f->lchild == p) f->lchild = p->rchild;
        else f->rchild = p->rchild;
        free(p);
    } else { // 情况4: 有左右子树，用中序前驱替换
        BSTNode *q = p->lchild;
        f = p;
        while (q->rchild != NULL) { // 找左子树最右下结点 (中序前驱)
            f = q;
            q = q->rchild;
        }
        p->key = q->key; // 用前驱值替换待删结点值
        // 删除前驱结点 q
        if (f->lchild == q) f->lchild = q->lchild;
        else f->rchild = q->lchild;
        free(q);
    }
    return bt;
}
