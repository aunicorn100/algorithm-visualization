核心思想（匹配例8.7）

  • 采用深度优先遍历（DFS）方法
  • path 存放路径，d 为当前路径长度（初值-1），l 为目标长度
  • 标记 u 为已访问，将 u 加入路径 path
  • 若 u == v 且 d == l，输出当前路径并回溯
  • 遍历 u 的邻接点 w，未访问则递归调用 PathlenAll
  • 递归结束后回溯：恢复 visited[u] 为未访问

核心算法代码（PathlenAll）

            
int visited[MAXV]; // 全局访问标记数组
// 输出从u到v的所有长度为L的简单路径
void PathlenAll(AdjGraph *G, int u, int v, int l, int path[], int d)
{  //d表示path中的路径长度，初始为-1
   int w, i;  ArcNode *p;
   visited[u] = 1; 		//置已访问标记
   d++; path[d] = u; 	//路径长度d增1，顶点u加入到路径中
   if (u == v && d == l) 	//找到一条长度为L的路径后输出
   {  printf("一条长度为%d的简单路径为:", l);
      for (i=0; i<=d; i++) printf("%d ",path[i]);
      printf("\n");
      visited[u] = 0; 	//回溯，恢复访问标记
      return; 		//返回继续寻找其他路径
   }
   p = G->adjlist[u].firstarc; 	//p指向顶点u的第一个邻接点
   while (p != NULL)
   {  w = p->adjvex; 		//相邻点的编号为w
      if (visited[w] == 0) 	//若w顶点未访问
          PathlenAll(G, w, v, l, path, d); //递归调用PathlenAll
      p = p->nextarc; 		//p指向顶点u的下一个邻接点
   }
   visited[u] = 0; 	//回溯，恢复访问标记
}
