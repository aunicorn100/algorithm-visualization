BFS核心思想

  • 从起始顶点出发，标记为已访问，输出顶点并入队
  • 队列非空时，出队顶点w，遍历其所有邻接顶点
  • 未访问的邻接顶点标记后入队，保证“先入先出”
  • 邻接表存储下，时间复杂度为 O(n+e)
  • 遍历思路：距离初始顶点越近越优先访问

邻接表BFS队列算法代码

            void BFS(AdjGraph *G, int v)
            {
                int w, i;              //定义邻接顶点w与循环变量i
                ArcNode *p;            //定义边指针p
                SqQueue *qu;           //定义环形队列指针
                InitQueue(qu);         //初始化队列
                int visited[MAXV];     //定义顶点访问标记数组
                for (i=0; i<G->n; i++) //访问标记数组初始化
                    visited[i] = 0;
                printf("%2d", v);      //输出被访问顶点的编号
                visited[v] = 1;        //置已访问标记
                enQueue(qu, v);        //v顶点进队
                while (!QueueEmpty(qu))//队不空循环
                {
                    deQueue(qu, w);     //出队一个顶点w
                    p = G->adjlist[w].firstarc; //指向w的第一个邻接点
                    while (p != NULL)   //查找w的所有邻接点
                    {
                        if (visited[p->adjvex] == 0) //若当前邻接点未被访问
                        {
                            printf("%2d", p->adjvex); //访问该邻接点
                            visited[p->adjvex] = 1;  //置已访问标记
                            enQueue(qu, p->adjvex);  //该顶点进队
                        }
                        p = p->nextarc;    //找下一个邻接点
                    }
                }
                printf("\n");
            }
            // 该算法的时间复杂度为 O(n+e)
