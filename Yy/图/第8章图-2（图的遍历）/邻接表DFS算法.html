DFS核心思想

  • 从起始顶点出发，标记为已访问，输出顶点
  • 递归访问其所有未被访问的邻接顶点
  • 一条路走到底，再回溯访问其他分支
  • 邻接表存储下，时间复杂度为 O(n+e)
  • 遍历思路：距离初始顶点越远越优先访问

邻接表DFS递归算法代码

            void DFS(AdjGraph *G, int v)
            {
                ArcNode *p; int w;              //定义边指针p与邻接顶点w
                visited[v] = 1;                 //置已访问标记
                printf("%d  ", v);              //输出被访问顶点的编号
                p = G->adjlist[v].firstarc;     //p指向顶点v的第一条边的边头结点
                while (p != NULL)               //遍历所有邻接边
                {
                    w = p->adjvex;              //取邻接顶点编号w
                    if (visited[w] == 0)        //若w顶点未被访问
                        DFS(G, w);              //递归访问邻接顶点w
                    p = p->nextarc;             //p指向顶点v的下一条边的边头结点
                }
            }
            // 该算法的时间复杂度为 O(n+e)
