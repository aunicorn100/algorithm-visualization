核心思想（匹配例8.8）

  • 简单回路：路径顶点不重复，起点与终点相同（长度≥2）
  • FindCyclePath(G, k)：入口，调用DFSPath(G, k, k, path, -1)
  • DFSPath：回溯DFS，搜索从u到v的所有简单路径
  • 当邻接点w==v且路径长度d>1时，输出回路 path + v
  • 回溯时恢复visited标记，保证所有路径被遍历

核心算法代码（匹配例8.8）

            
int visited[MAXV]; // 全局访问标记数组
// 输出经过顶点k的所有回路
void FindCyclePath(AdjGraph *G, int k)
{  int path[MAXV];
   DFSPath(G, k, k, path, -1); // 起点和终点都是k
}

// 搜索从u到v的所有简单路径，d为当前路径长度
void DFSPath(AdjGraph *G, int u, int v, int path[], int d)
{  int w, i; ArcNode *p;
   visited[u] = 1;      // 标记u为已访问
   d++; path[d] = u;    // 将u加入路径
   p = G->adjlist[u].firstarc; // p指向u的第一个邻接点
   while (p != NULL)
   {  w = p->adjvex;    // w为u的邻接点
      if (w == v && d > 1) // 找到回路，输出
      {  printf("  ");
         for (i=0; i<=d; i++) printf("%d ", path[i]);
         printf("%d \n", v);
      }
      if (visited[w] == 0) // 若w未访问，递归访问
          DFSPath(G, w, v, path, d);
      p = p->nextarc;    // p指向下一个邻接点
   }
   visited[u] = 0;      // 回溯，恢复u为未访问
}
