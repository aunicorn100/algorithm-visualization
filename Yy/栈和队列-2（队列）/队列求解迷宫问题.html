BFS 算法

算法简介

广度优先搜索（BFS）是一种图遍历算法，适合寻找最短路径。

核心思想：

  1. 从起点开始，将其加入队列
  2. 取出队首节点，检查是否为终点
  3. 将所有未访问的邻居加入队列
  4. 重复步骤2-3，直到找到终点

为什么用队列？

队列的先进先出特性保证了按层次探索，首次到达终点的路径就是最短路径。

function solveMaze() {
    // 初始化队列，加入起点

    while (队列不为空) {
        // 出队一个位置
        let current = 队列.shift();

        // 找到终点，回溯路径
        if (current是终点) {
            return 回溯路径(current);
        }

        // 探索四个方向
        for (方向 in [上,右,下,左]) {
            let next = 计算新位置;

            if (next是有效路径且未访问) {
                记录前驱节点
                加入队列
                标记为已访问
            }
        }
    }
}
