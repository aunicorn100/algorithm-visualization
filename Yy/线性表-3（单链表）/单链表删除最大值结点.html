算法思路

核心：通过两对同步指针遍历链表，追踪最大值结点及其前驱

  • p/pre：遍历指针（p当前结点，pre前驱）
  • maxp/maxpre：最大值追踪指针
  • 一次遍历完成查找与删除，时间复杂度O(n)

算法可视化演示

算法代码（执行行高亮） 

// 删除单链表中值最大的结点（最大值唯一）
void delmaxnode(LinkNode *&L) {
    LinkNode *p = L->next, *pre = L;  // 遍历指针：p当前结点，pre前驱
    LinkNode *maxp = p, *maxpre = pre;// 最大值指针：maxp最大值结点，maxpre其前驱

    while (p != NULL) {                // 遍历整个链表
        if (maxp->data < p->data) {    // 找到更大值，更新最大值指针
            maxp = p;                  // 更新最大值结点
            maxpre = pre;              // 更新最大值结点的前驱
        }
        pre = p;                       // 前驱指针后移
        p = p->next;                   // 当前指针后移
    }
    maxpre->next = maxp->next;         // 删除最大值结点
    free(maxp);                        // 释放最大值结点内存
}
