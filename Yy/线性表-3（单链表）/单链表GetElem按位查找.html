GetElem算法代码 

bool GetElem(LinkNode *L, int i, ElemType &e)
{
    int j = 0;
    LinkNode *p = L; // p指向头结点，j置为0

    while (j < i && p != NULL)
    {
        j++;
        p = p->next;
    }

    if (p == NULL) // 不存在第i个数据结点
        return false;
    else
    {
        e = p->data;
        return true;
    }
}

算法步骤详解

1

初始化

设置j=0，p指向头结点

2

查找循环

当j<i且p≠NULL时，j++，p=p->next

3

边界检查

检查p是否为NULL，判断第i个结点是否存在

4

返回结果

如果找到则返回数据，否则返回false
