算法核心思想

  • 从头结点开始顺序查找，比较每个结点的数据域
  • 找到目标值返回其逻辑序号，未找到返回0
  • 时间复杂度为O(n)，不支持随机访问

算法代码

            int LocateElem(LinkNode *L, ElemType e)
            {
                int i = 1;                      //i置为1
                LinkNode *p = L->next;          //p指向开始结点
                while (p != NULL && p->data != e) //查找data值为e的结点
                {
                    p = p->next;                //移动到下一结点
                    i++;                        //序号增1
                }
                if (p == NULL)                  //不存在元素值为e的结点
                    return 0;                   //返回0
                else
                    return i;                   //返回其逻辑序号i
            }
