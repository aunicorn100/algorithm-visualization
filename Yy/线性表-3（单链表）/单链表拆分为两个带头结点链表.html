单链表拆分算法可视化

问题描述

【例2.6】假设有一个带头结点的单链表L=(a₁,b₁,a₂,b₂,...,aₙ,bₙ)。设计算法拆分为两个带头结点链表：

L₁=(a₁,a₂,...,aₙ)（复用L的头结点），L₂=(bₙ,bₙ₋₁,...,b₁)

算法思路

通过两种经典建表法实现拆分：

  • L₁用 尾插法 保持a类结点顺序
  • L₂用 头插法 实现b类结点逆序

算法分析

  • 时间复杂度：O(n)（单次遍历）
  • 空间复杂度：O(1)（仅用指针变量）
  • 核心技术：尾插法（L₁）、头插法（L₂）
  • 优点：高效简洁，无需额外存储

算法代码

// 单链表拆分函数
void SplitList(LinkNode *&L, LinkNode *&L1, LinkNode *&L2) {
    L1 = L;             // L1复用原链表L的头结点
    L2 = (LinkNode *)malloc(sizeof(LinkNode));  // 为L2创建新头结点
    L2->next = NULL;    // 初始化L2为空链表（头结点后无数据）

    LinkNode *p = L->next;  // p指针：遍历原链表的工作指针（从首结点开始）
    L1->next = NULL;    // 初始化L1为空链表（头结点后无数据）
    LinkNode *q;        // q指针：临时存储当前待处理的结点
    LinkNode *r1 = L1;  // r1指针：始终指向L1的尾结点，用于尾插

    int i = 0;          // 计数器：标记当前结点位置（1开始），判断a/b类
    while (p != NULL) { // 遍历原链表，直到所有结点处理完毕
        i++;
        q = p;          // 保存当前结点到q
        p = p->next;    // p指针后移，避免后续操作丢失后继结点

        if (i % 2 == 1) {  // 奇数位置：a类结点，尾插法插入L1
            r1->next = q;  // 将当前结点接入L1尾部
            r1 = q;        // r1更新为新的尾结点
            r1->next = NULL; // 确保L1尾部指向NULL
        } else {         // 偶数位置：b类结点，头插法插入L2
            q->next = L2->next; // 当前结点指向L2头结点的后继
            L2->next = q;        // L2头结点指向当前结点（完成头插）
        }
    }
}
