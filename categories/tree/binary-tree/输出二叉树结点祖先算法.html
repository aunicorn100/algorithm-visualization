例题7.16：输出值为x的结点的所有祖先

基于二叉链存储结构的递归算法实现与可视化

问题描述

假设二叉树采用二叉链存储结构，设计一个算法输出值为x的结点的所有祖先（结点x唯一）。

算法实现一：递归回溯法 


bool ancestor(BTNode *b, ElemType x)
{
    if (b == NULL)
        return false;
    // 如果当前结点的孩子是目标结点x，则当前结点是祖先
    else if ((b->lchild != NULL && b->lchild->data == x) || 
             (b->rchild != NULL && b->rchild->data == x))
    {
        printf("%c ", b->data); // 输出祖先结点
        return true;
    }
    // 递归在左、右子树中查找，如果找到，当前结点也是祖先
    else if (ancestor(b->lchild, x) || ancestor(b->rchild, x))
    {
        printf("%c ", b->data); // 输出祖先结点
        return true;
    }
    else
        return false; // 未找到
}

特点：从目标结点的父节点开始，由下至上输出祖先。

算法实现二：路径记录法 


bool ancestor1(BTNode *b, ElemType x, ElemType path[], int d)
{
    bool find;
    if (b == NULL)
        return false;
    d++;
    path[d] = b->data; // 将当前结点加入路径
    if (b->data == x) // 找到目标结点
    {
        // 输出路径中除目标结点外的所有结点（即祖先）
        for (int i = 0; i < d; i++)
            printf("%c ", path[i]);
        return true;
    }
    // 递归在左子树中查找
    find = ancestor1(b->lchild, x, path, d);
    if (find)
        return true;
    else // 左子树未找到，在右子树中查找
        return ancestor1(b->rchild, x, path, d);
}

特点：记录从根结点到目标结点的完整路径，然后输出路径上的祖先，由上至下输出。
