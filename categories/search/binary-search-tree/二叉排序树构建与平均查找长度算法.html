2. 平均查找长度 (ASL)

查找成功时的ASL：

ASL成功 = (1×1 + 2×2 + 3×3 + 3×4 + 2×5 + 1×6) / 12 = 3.5

查找不成功时的ASL：

ASL不成功 = (1×2 + 3×3 + 4×4 + 3×5 + 2×6) / 13 ≈ 4.15
