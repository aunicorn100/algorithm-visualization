算法核心思想

  • 使用两个指针i和j分别指向主串和模式串的当前位置
  • 当字符匹配时，两个指针同时向前移动
  • 当字符不匹配时，主串指针移动到下一个位置，模式串指针重置为0
  • 时间复杂度为O(m*n)，其中m为主串长度，n为模式串长度
  • 这是BF算法的一种变体实现，与经典版本略有不同

BF算法代码

            int BF(SqString s, SqString t)
            {
                int i = 0, j = 0;                    //定义变量
                while (i < s.length && j < t.length) //循环匹配
                    if (s.data[i] == t.data[j])      //字符匹配
                    {
                        i++;                         //主串和子串依次匹配下一个字符
                        j++;
                    }
                    else                             //字符不匹配
                    {
                        i++;                         //主串从下一个位置开始匹配
                        j = 0;                       //子串从头开始匹配
                    }
                if (j >= t.length)                   //匹配成功
                    return i - t.length;             //返回匹配的第一个字符的下标
                else
                    return -1;                       //模式匹配不成功
            }
