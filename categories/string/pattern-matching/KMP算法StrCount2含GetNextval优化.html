算法核心思想

  • 使用改进的KMP算法（nextval优化）高效查找模式串在主串中的所有出现位置
  • 通过预处理模式串生成nextval数组，避免不必要的字符比较
  • nextval优化通过递归跳过相同字符，进一步减少比较次数
  • 当字符不匹配时，根据nextval数组移动模式串指针，提高匹配效率
  • 时间复杂度为O(m+n)，其中m为主串长度，n为模式串长度
  • StrCount2函数用于统计模式串在主串中出现的次数
  • GetNextval函数用于计算模式串的nextval数组

改进的KMP算法代码

            int StrCount2(SqString s, SqString t) //利用改进的KMP算法求t在s中出现的次数
            {
                int i = 0, j = 0, count = 0;       //定义变量
                int nextval[MaxSize];              //定义nextval数组
                GetNextval(t, nextval);            //计算nextval数组
                while (i < s.length && j < t.length) //循环匹配
                    if (j == -1 || s.data[i] == t.data[j]) //字符匹配
                    {
                        i++;                         //主串和子串依次匹配下一个字符
                        j++;
                    }
                    else                             //字符不匹配
                        j = nextval[j];              //子串根据nextval数组回退
                if (j >= t.length)                   //匹配成功
                {
                    count++;                         //计数器加1
                    j = 0;                           //重置j为0，继续匹配
                }
                return count;                        //返回匹配次数
            }
            
            void GetNextval(SqString t, int nextval[])
            {
                int j, k;
                int next[MaxSize];                 //临时next数组
                j = 0; k = -1; next[0] = -1;       //初始化next数组
                while (j < t.length - 1)
                    if (k == -1 || t.data[j] == t.data[k])
                    {
                        j++; k++;
                        next[j] = k;
                    }
                    else
                        k = next[k];
                }
                //根据next数组计算nextval数组
                for (j = 0; j < t.length; j++)
                    if (next[j] == -1)
                        nextval[j] = -1;
                    else if (t.data[j] == t.data[next[j]])
                        nextval[j] = nextval[next[j]];
                    else
                        nextval[j] = next[j];
            }
