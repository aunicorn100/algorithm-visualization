算法核心思想

  • 使用三个指针i、j、k分别控制主串起始位置、模式串位置和主串比较位置
  • 外层for循环控制主串起始位置i，内层for循环同时比较主串和模式串
  • 当字符匹配时，k和j同时向前移动
  • 当字符不匹配时，主串起始位置i移动到下一个位置
  • 当完整匹配模式串时，计数器count增加
  • 时间复杂度为O(m*n)，其中m为主串长度，n为模式串长度

int Substrcount1(char *s, char *t)
//利用BF算法求t在s中出现的次数
{  int i, j, k, count = 0;
   int m = strlen(s);
   int n = strlen(t);
   for (i = 0; i <= m - n; i++)
   {  for (k = i, j = 0; k < m && j < n && s[k] == t[j]; k++, j++);
      if (j == n)
         count++;
   }
   return (count);
}
