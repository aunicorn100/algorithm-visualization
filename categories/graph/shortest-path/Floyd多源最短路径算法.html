教材8.5.3 核心要点

  • 求每对顶点之间最短路径
  • 递推公式：Aₖ[i,j] = min(Aₖ₋₁[i,j], Aₖ₋₁[i,k]+Aₖ₋₁[k,j])
  • 依次以 k=0~n-1 为中间点
  • 时间复杂度：O(n³)
  • 不适合负权回路图

Floyd 算法代码（完全同PPT）


#define INF 32767		// 表示无穷大
#define MAXVEX 10		// 最大顶点数

void Floyd(MatGraph g)		// 多源最短路径
{  int A[MAXVEX][MAXVEX];	// 路径长度矩阵
   int path[MAXVEX][MAXVEX];	// 路径前驱矩阵
   int i, j, k;

   // 初始化 A 与 path
   for (i=0; i<g.n; i++)
      for (j=0; j<g.n; j++)
      {  A[i][j] = g.edges[i][j];
         if (i!=j && g.edges[i][j]<INF)
             path[i][j] = i;
         else
             path[i][j] = -1;
      }

   // 三层循环迭代
   for (k=0; k<g.n; k++)		// 依次以k为中间点
      for (i=0; i<g.n; i++)	// 所有起点i
         for (j=0; j<g.n; j++)	// 所有终点j
             if (A[i][j] > A[i][k] + A[k][j])
             {  A[i][j] = A[i][k] + A[k][j];
                path[i][j] = path[k][j];
             }
}
