教材核心要点

  • 不带权无向连通图中，BFS 遍历的最后一个出队顶点即为最远顶点
  • 用队列保存待访问顶点，visited 数组标记已访问顶点
  • 时间复杂度：O(n+e)
  • 本质：BFS 按层遍历，最后一层顶点即为最远顶点

无向连通图结构可视化（环形，起点与最远顶点高亮）

Maxdist 算法代码（完全同PPT）


int Maxdist(ALGraph *G, int v)
{  ArcNode *p;
   int Qu[MAXV], front=0, rear=0;    //队列及队头、尾指针
   int visited[MAXV], i, j, k;
   for (i=0;i<G->n;i++)            //初始化访问标志数组
       visited[i]=0;
   rear++;Qu[rear]=v;               //顶点v进队
   visited[v]=1;                   //标记v已访问

   while (rear!=front)             //队列不空循环
   {  front=(front+1)%MAXV;         //顶点出队
      k=Qu[front];
      p=G->adjlist[k].firstarc;     //找第一个邻接点
      while (p!=NULL)               //所有未访问过的邻接点进队
      {  j=p->adjvex;               //若j未访问过
         if (visited[j]==0)
         {  visited[j]=1;           //将顶点j进队
            rear=(rear+1)%MAXV;Qu[rear]=j;
         }
         p=p->nextarc;              //找下一个邻接点
      }
   }
   return k;                       //返回最远顶点k
}
