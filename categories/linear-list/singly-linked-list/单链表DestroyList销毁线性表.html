算法概述

用同步指针（pre/p）遍历链表，释放所有结点内存，包括自动添加的头结点。

时间复杂度： O(n)
空间复杂度： O(1)

算法代码 

// 销毁线性表DestroyList(L)
void DestroyList(LinkNode *&L)
{   
    LinkNode *pre=L,*p=L->next;
    while(p!=NULL)
    {  
        free(pre);	// 释放pre结点（含头结点）
        pre=p;		// 指针后移
        p=pre->next;
    }
    free(pre); // 释放尾结点
}

销毁步骤

1

初始化指针

pre=L（指向自动添加的头结点），p=L->next（第一个数据结点）

2

循环遍历

进入while循环，先释放头结点，再依次释放数据结点

3

释放尾结点

p=NULL时退出循环，free(pre)释放尾结点
